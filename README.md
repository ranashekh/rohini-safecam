# Rohini Safecam Web Application

A professional web application for Rohini Safecam, a leading security solutions provider in West Bengal.

## Features
- High-density design theme
- Full surveillance portfolio
- Service inquiry system
- Comprehensive technical highlights
- Responsive design for mobile and desktop

## Tech Stack
- React 19
- TypeScript
- Tailwind CSS 4
- Lucide React Icons
- Motion (for animations)
- Vite

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
